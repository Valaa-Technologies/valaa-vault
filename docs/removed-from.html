Ontology of the secondary `removed-from` fields of the primary valospace fields which have them (ie. those which statefully track entries that have been removed from them).

This document is part of the vault workspace @valos/kernel (of domain @valos/kernel) which has the description: `ValOS common infrastructure tools and libraries monorepository.`.

Some container fields maintain a hidden list of entries that have been removed from them, f.ex. for expressing removals from ghost fields which are empty as they inherit their entries from the prototype.

§ The VRemovedFrom valospace namespace

'VRemovedFrom' removed-from namespace provides vocabulary and definitions for valospace resource removed-from fields.
